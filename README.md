## Tech Stack

- HTML5
- CSS with CSS Modules
- TailwindCSS
- TypeScript
- React.JS
- React Router
- Vite

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.
